# Employee - App

---
## Installation Guide

### 1. Clone Repository
```bash
git clone https://github.com/jalilnawawi/employee-app.git
```
### 2. Open repository using your editor or IDEA
### 3. Run file `EmployeeAppApplication.java`
### 4. Open Employee-App UI by this URL 
```bash
http://localhost:8080/employees
```
